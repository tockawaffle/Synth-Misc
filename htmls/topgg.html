Commands

With Synth Bot, you can interact with specific commands or start a chat using a subject and model ofyour choice:

  • Shows a list of categories of avaiable commands: /help
  • * Sets the category for the specified model: /config category <tBing | GPT> <#Category>
  • Sets the language for the bot (Per User): /config language <English | Portugues>
  • * Will provide help while using the start-chat command: /start-chat help
  • * Starts a chat using the specified subject and model: /start-chat assistants <subject> <model>
  • * Generates an image with the given prompt and size, using the DALL-E model: /image dall-e <prompt> <size>
  • * Will provide some more information on what the optional parameters do on each whisper command. /whisper help <translate | transcribe>
  • * Transcribes a video from the given URL, using the specified language, defaulting to english, you can also use prompting to guide the model: /whisper transcribe <url> <prompting?> <language?>
  • * Translates a video to english from the given URL, you can use prompting to guide the model: /whisper translate <url> <prompting?>
  • * Asks a question to the model: /ask-me <question> <model>

You can also DM the bot to start a chat without having to specify a subject or model!

* - Only works on servers. ? - Optional parameter.
